Meu Kit de Ferramentas Digitais

HTML

É a linguagem usada para criar páginas da web. Ela estrutura o conteúdo como textos, imagens, vídeos, links e botões para que o navegador saiba como exibir as informações.

CSS

É a linguagem usada para estilizar páginas web ou seja, dar cores, tamanhos, fontes, margens, posicionamento e muito mais ao conteúdo feito em HTML.

JavaScript

JavaScript é uma linguagem essencial no desenvolvimento web moderno. Em conjunto com HTML (estrutura) e CSS (estilo), o JavaScript adiciona funcionalidade e interatividade, tornando os sites mais dinâmicos e responsivos.
